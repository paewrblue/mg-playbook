The iron gate — never violated
MG is never a standalone buy reason. The complete stack is 7 lenses: durable business · catalyst · fundamental · smart money · insider flow · technical · money game. No researched fundamental thesis = information only, no position. MG times entries inside a zone the fundamentals already approve — it never widens one. A perfect footprint on a broken business is the operator's trade, not ours.

1What the machine measures — the primitives

Input: 10 years of daily OHLCV per name. Everything derives from one number and three indicators:

turnover[i] = close[i] × volume[i]        # baht traded that day — the operator's unit (baht, not shares)
RSI(14) · MACD(12/26/9) · OBV             # momentum, momentum-of-momentum, and the sponsorship meter

Windows are trading days (td): 20td ≈ 1 month, 60td ≈ 1 quarter, 250td ≈ 1 year. The analysis window BASE = 60td is where a game would currently be built; the 60td before it is the "what it used to trade" baseline. Why baht and not shares: the operator's economics — inventory cost, markup profit, exit capacity — are all denominated in money. His footprint is a money footprint.

A stock's life in this framework begins asleep. Act 1 ↓

2Act 1 · The sleep — dormancy

Dormancy = the median daily turnover of the year before the base window (median ⇒ one-day spikes can't fake it): 
dorm_med = median( turnover[−310 : −60] )
class    = DEEP-DORMANT if dorm_med < ฿5M/d  ·  quiet if < ฿30M/d  ·  already-liquid otherwise
awaken_x = avg(turnover[−60:]) / dorm_med      # how many times its old self it now trades
Deep dormancy is the best starting condition we measured — games born from it paid +42%/ride vs +3.6% for already-liquid names. The mechanism is identification: when a dead stock suddenly trades, there is only one kind of buyer it can be; in a liquid name the same baht vanish into noise.

One day, volume arrives while price still sleeps. Act 2 ↓

3Act 2 · The awakening — the radar (watch-only, never an entry)

ping = sma(turnover[−20:]) ≥ 5× median(turnover[−270:−20])   # volume woke up…
       and −15% < price_chg_20d < +15%                        # …while price still sleeps
       and sma(turnover[−20:]) × 20 ≥ ฿50M                    # and the new tape is already game-worthy
The extreme case caught live: a construction name went ฿43K/day → ฿4.5M/day — a 104× awakening — with price still flat. The radar's product is lead time: 53% of deep-dormant pings produce a confirmed ignition ~23 trading days later. That month is for the fundamental homework (the iron gate's dossier and value zone), never for buying — buying the ping itself was tested: it loses to waiting. Front-running accumulation was tested too: a 34% hit rate. Thai operators don't telegraph.

Then the operator commits real money and price leaves the base. Act 3 ↓

Worthiness is a monthly capacity question — "could an operator size a real game in this tape?" — never a weekly intensity question: ฿50M/week on a dormant name means the marketing already worked; peak burn is a late-game property. A chips proxy estimates inventory already collected: chips ≈ 0.5 × Σ max(turnover − old_baseline, 0).

One entry, at ignition, full size — averaging down was tested and rejected (losers always fill the add; 61% of winners never dip 10%). The EMA20 pullback is only the second-chance door if the ignition was missed. After a signal the scanner cools down 40td and holds one position per name — which is why ~860 names × 10 years produce 1,577 discrete signals, not thousands.
Execution — the reality tax, measured on 1,955 rides
The signal prints after the close → enter at the next open. Measured cost ≈ 1 point/trade; ignition signals do not gap (median overnight gap +0.0% — the operator walks price, he doesn't jump it). Skip any entry gapping >+5% over the signal close (25 of 1,955 cases — free insurance), and work size at ≤10% of the day's turnover.

You're in. Now the machine tells time — how old is this game, and which phase is it in? Act 4 ↓

5Act 4 · The ride — the game clock & the phase read

The game clock — finding the game's birthday

active[i] = 20d_MA(turnover)[i] > max( 1.5 × median(turnover[i−270:i−20]), ฿2.5M/d )
origin    = walk BACK from today through the active run, tolerating gaps ≤ 20td
base      = min(close) within origin ± 10td;   age = today − origin;   gain = close/base − 1

# legs: new-high runs separated by ≥10% pullbacks — more legs = older game
# divergence (the operator's exit fingerprint), across the last two leg peaks p1 < p2:
div = close[p2] > close[p1]  and  ( RSI[p2] < RSI[p1] − 3  or  MACDhist[p2] < MACDhist[p1] )
The absolute ฿2.5M/day floor matters: on a ฿43K/day baseline the 1.5× relative bar trips on noise months before the real game. Requiring game-worthy baht moved one measured origin from a phantom January to the Q1 earnings print in May — where the eye said the game actually began.

Game-aware technicals (not textbook)
RSI > 70 = STRENGTH in a markup, not a sell — sell the divergence, never the overbought. RSI < 45 = games rarely resume without a new ignition. EMA20 = the operator's markup rail (healthy games hold it ≥70% of days; a break on running volume = the exit bell). EMA10 catches half of all winner shakeout lows — the first 2 weeks after entry are the shakeout window: expect it, don't react to it. OBV rising with price = real money staying; price up on flat OBV = hollow markup, fragile.

Every ride ends. The machine decides when — you don't. Act 5 ↓

6Act 5 · The exit — the rider state machine (no profit target, ever)

state = {armed: False, below: 0}
for each close c after entry, until day 250 (LEASH):
    if c ≤ entry × 0.75:                exit("stop")     # disaster stop — close-based, never tighter
    if not armed and c ≥ entry × 1.15:  armed = True     # arming SELLS NOTHING
    if armed:
        below = below+1 if c < EMA50 else 0
        if below ≥ 2:                   exit("trail")    # 2 consecutive closes under the EMA50
    if day == 126 and c ≤ entry:        exit("rotate")   # dead-money rotation at 6 months (tested: free)
if still open at day 250:               exit("leash")    # the 1-year leash for Thai names

The operator's +20–30% markup math screens games; it is never our exit. The rotation ladder does the time-capping: 3 months = review only · 6 months = rotate if flat · 12 months = the leash. A durable GULF/DELTA-class business earns a structural hold at the 1-year review through the business lens, not through an uncapped-vs-capped rule.
In validation — the heaviness exit ("the selling climate")
The newest candidate, currently on live trial against the trail: maximum volume + minimum progress, at the highs — after arming, two days within two weeks where 20d turnover runs ≥1.5× the entry level while 15-day price progress is ≤+2% and price sits within 10% of the ride high = the operator distributing into peak enthusiasm → sell into the heavy top instead of waiting for the trend break. Backtested on 882 rides: it beats the EMA50 trail in every liquidity-ON year (median trade doubles; keeps ~92% of the ride's peak vs 84%; keeps MORE of the monster rides — the trail's mid-campaign whipsaws were the real tail-killer) and loses in every tightening year (games die quietly there: no climax ever prints, and without a trail the rides bleed to the stop). The law: heaviness is a fair-weather exit; the trail is the all-weather seatbelt — exit merit flips at the same regime boundary that governs entries. Its first live-season test fired this month and beat the trail by ~8 points; it becomes the rule only if the forward validation keeps crowning it.

The cap boundary — when it's not a money game at all
The playbook is structurally small-cap. Tested up the ladder with size-scaled floors (฿200M and ฿1B/month for mid and large names): the machinery still fires, but the edge dissolves — roughly 13 signals a year instead of 145, expectancy a third of the small-cap funnel's, and the ≥+50% tail class simply absent above ~฿30M/day of resting liquidity. The three ingredients the game needs — a float one player can control, a dormancy deep enough that awakening identifies the buyer, a crowd that must be manufactured — don't exist upstairs. A volume surge on a large cap is an institutional rotation (the DELTA-2022 / telecom-merger-2021 class): win rate ~62%, modest gains, less regime-fragile — a real signal, but it belongs to breakout/trend playbooks with fundamental sizing, never to money-game economics. Rule of thumb: median turnover above ฿30M/day = rotation lane, different weapon.

9Hard rules & the graveyard

Hard rules
1 · No position in a pure money game — no researched fundamental case, no entry   2 · No entry while on SET Cash Balance / Trading Alert   3 · Never chase markup >20% above the approved zone   4 · A distribution read on a held name = trim review the same week   5 · ฿50M is a floor — sub-scale games are retail roulette   6 · Every call is written down, dated, and scored later
Tested and rejected — stays dead
Tight stops / EMA20 trails (operator shakeouts harvest them) · averaging down at −10% (adverse selection) · profit targets and hard caps, including banking the operator's +30% and 2x/3x brackets (the tail pays for everything) · valuation-stretch & sell-the-news auto-exits (a +381% ride looks "farfetched" the whole way up — in a money game the rally is SUPPOSED to exceed sane valuation; that's the product) · forced 3-month rotation · preferring the pullback entry · the ฿50M-in-5-days worthiness window (peak burn is a LATE-game property; the gate drops the single best class — the gradual-burn games) · index-trend regime gates (TDEX≥200d at any resolution: the decade's best rides ignite below the 200d) · Fibonacci retracements (winner retracements scatter 0–67%; no clustering at 38.2/50/61.8) · Elliott Wave as formal method (unfalsifiable; its useful core — leg counting and divergence-across-peaks — is already mechanical in the game clock)

What this means
The detector alone is worth ~nothing; the funnel is everything. A screener is a radar, never a signal service. The liquidity regime is the single biggest lever (~21 points per trade): when the tide turns, the correct number of new games is zero — in the tightening years the same signal fired 500+ times and refusing every one of them was the single most profitable decision in the record. The exit parameters sit on a plateau (stop 20/25/30, trail EMA30/50/100 all within a point on identical entries) — the edge comes from selection, never from exit tuning.
The honesty rules the test harness enforces
Full-universe or it didn't happen — 30-name smoke tests reversed the full-universe verdict twice. · Ex-ante only — every conditioning variable is computable at the entry date. · Survivorship stated — delisted deaths are absent, so absolute expectancies are optimistic; relative comparisons stand. · A forward book is the referee — every live signal is frozen at its scan-day price against a control group of non-qualifying ignitions and re-scored weekly; if the live funnel stops beating its control, everything recalibrates. · Portfolio truth beats trade truth — simulating one book through the same 74 signals showed the edge is a fat-tailed distribution: a book sized to stay in the game for the next signal compounds it; all-in concentration (which could only take 8 of 74 signals) turned the same edge into a coin-flip book with twice the drawdown.
